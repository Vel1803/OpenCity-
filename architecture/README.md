# Architecture

This folder stores the visual architecture diagrams for OpenCity:

- `System_Architecture.png` — overall system components and data flow
- `Workflow.png` — end-to-end project → conflict-check → approval → maintenance flow
- `AI_Workflow.png` — how the AI conflict-detection/recommendation pipeline works
- `Database_ER.png` — entity-relationship diagram for the database
- `Digital_Twin.png` — architecture of the live digital twin layer
