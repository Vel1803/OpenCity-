# OpenCity — Frontend

React / Next.js frontend for the OpenCity platform, styled with Tailwind CSS.

## Planned Modules
- **Government Dashboard** — project coordination, budgets, asset registry, executive view
- **Citizen Portal** — project tracking, road closure updates, issue reporting
- **Maps** — interactive city map (OpenStreetMap + Leaflet) showing the digital twin
- **Login** — Supabase Auth–based authentication for departments and citizens
- **Reports** — AI-generated reports and analytics views
